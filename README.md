# Sales Performance Analysis Dashboard

## Overview

This project analyzes sales performance data to identify revenue trends, customer purchasing behavior, and business growth opportunities using SQL and Power BI.

The dashboard provides business insights through KPI monitoring, trend analysis, and interactive visual reports.

---

## Tools Used

* Power BI
* SQL
* Microsoft Excel
* Python (Basic)

---

## Features

* KPI Tracking Dashboard
* Revenue Trend Analysis
* Product Performance Insights
* Regional Sales Analysis
* Customer Contribution Analysis
* Interactive Data Visualization

---

## Key Insights

* Identified top-performing product categories
* Tracked monthly sales growth trends
* Evaluated region-wise revenue distribution
* Analyzed customer purchasing patterns
* Compared profit contribution across regions
